How it works

Three commands. Your project, deployed.

Point Neo at a fresh VPS and it handles everything. It reads your existing docker-compose.yml — no extra config needed.

01

Initialize your server

Neo connects over SSH, installs Docker, starts Caddy as a reverse proxy, and creates the server state file.

$ neo init root@159.65.100.42
02

Install an app

Pick from built-in templates — Ghost, Plausible, Umami, WordPress, and more. Services and routing are bundled.

$ neo install ghost --domain blog.me
03

Deploy your code

Ship your own Dockerfile project. Neo auto-reads env vars from your docker-compose.yml or .env file. Zero-downtime deploys.

$ neo deploy . --domain app.me
Features

Everything you need.
Nothing you don't.

A focused CLI that covers the full deployment lifecycle without platform sprawl.

Multi-server management

Add, switch, and manage multiple servers from a single config. Each server keeps its own isolated state.

Automatic HTTPS

Caddy handles TLS certificates automatically. Assign a domain and SSL is provisioned in the same step.

App templates

One-command install for Ghost, Plausible, Umami, WordPress, Gitea, and more. Databases and volumes included.

Deploy your own code

Build from a local Dockerfile, transfer images over SSH, and route traffic — no registry or CI pipeline required.

Background workers

Run queue workers, schedulers, or any background process alongside your app. Declare them in .neo.yml and they deploy automatically.

Persistent volumes

Declare volumes in .neo.yml to persist databases, uploads, and storage across redeploys. Data survives every update.

Live logs & status

Stream container logs in real time, check running status, and inspect apps — all through the same SSH tunnel.

Env & config management

Auto-reads docker-compose.yml and .env files. Set, import, or update env vars with neo env — container restarts automatically.

Zero-downtime deploys

Blue-green deployment swaps traffic to the new container only after health checks pass. Your old version keeps running until the switch.
