# n8n

Workflow automation tool that lets you connect anything to everything.

## Quick Facts

- **Image**: `n8nio/n8n:1.72`
- **Port**: 5678
- **Services**: PostgreSQL 16 (bundled)
- **Min RAM**: 1GB

## Post-Install

1. Visit your domain to create your admin account on first launch.
2. Build workflows using the visual editor with 400+ integrations.
3. Webhooks are available at `https://<your-domain>/webhook/`.
