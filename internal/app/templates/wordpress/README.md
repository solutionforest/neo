# WordPress

The world's most popular content management system, powering over 40% of all websites.

## Quick Facts

- **Image**: `wordpress:6.7-apache`
- **Port**: 80
- **Services**: MySQL 8.0 (bundled)
- **Min RAM**: 512MB

## Post-Install

1. Visit `https://<your-domain>/wp-admin/` to run the setup wizard and create your admin account.
2. Choose your site language, title, and admin credentials during setup.
3. Install themes and plugins from the admin dashboard.
