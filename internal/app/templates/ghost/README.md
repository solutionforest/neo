# Ghost

Professional publishing platform for blogs, newsletters, and paid memberships.

## Quick Facts

- **Image**: `ghost:5-alpine`
- **Port**: 2368
- **Services**: MySQL 8.0 (bundled)
- **Min RAM**: 512MB

## Post-Install

1. Visit `https://<your-domain>/ghost/` to create your admin account.
2. Configure your site title, staff users, and newsletter settings from the admin panel.
3. Ghost stores all content and themes in a persistent volume.
