# Umami

Simple, fast, privacy-focused web analytics as an alternative to Google Analytics.

## Quick Facts

- **Image**: `ghcr.io/umami-software/umami:postgresql-v2.15.0`
- **Port**: 3000
- **Services**: PostgreSQL 16 (bundled)
- **Min RAM**: 512MB

## Post-Install

1. Log in with the default credentials: **admin** / **umami**.
2. Change the default password immediately after first login.
3. Add your websites and embed the tracking script to start collecting data.
