# Miniflux

Minimalist and opinionated RSS feed reader with a clean interface.

## Quick Facts

- **Image**: `miniflux/miniflux:2.2.5`
- **Port**: 8080
- **Services**: PostgreSQL 16 (bundled)
- **Min RAM**: 256MB

## Post-Install

1. You will be prompted for admin username and password during installation.
2. Log in and start adding RSS feed URLs to follow.
3. Miniflux supports Fever and Google Reader compatible APIs for third-party clients.
