# Plausible Analytics

Privacy-friendly, lightweight alternative to Google Analytics. No cookies, fully GDPR compliant.

## Quick Facts

- **Image**: `ghcr.io/plausible/community-edition:v2.1.4`
- **Port**: 8000
- **Services**: PostgreSQL 16 + ClickHouse 24.3 (both bundled)
- **Min RAM**: 2GB

## Post-Install

1. Visit your domain to create your admin account on first launch.
2. Add the Plausible tracking script to your website's `<head>` to start collecting data.
3. All analytics data is stored locally -- no data is sent to third parties.
