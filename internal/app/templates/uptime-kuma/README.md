# Uptime Kuma

Self-hosted monitoring tool with a beautiful status page and notification support.

## Quick Facts

- **Image**: `louislam/uptime-kuma:1`
- **Port**: 3001
- **Services**: None (uses SQLite internally)
- **Min RAM**: 256MB

## Post-Install

1. Visit your domain to create your admin account on first launch.
2. Add monitors for your websites, APIs, or TCP services.
3. Configure notifications via Slack, Telegram, Discord, email, and 90+ other services.
