# Chatwoot

Open-source customer engagement platform with live chat, email, and social media support.

## Quick Facts

- **Image**: `chatwoot/chatwoot:v3.14.0`
- **Port**: 3000
- **Services**: PostgreSQL 16 + Redis 7 (both bundled)
- **Min RAM**: 2GB

## Post-Install

1. Visit your domain to create your admin account on first launch.
2. Set up your inbox channels (website live chat, email, Facebook, Twitter, etc.).
3. Add the Chatwoot widget script to your website for live chat support.
